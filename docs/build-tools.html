Build Tools

We use Laravel Mix as the main build tool in this template. Laravel Mix provides a clean, fluent API for defining basic webpack build steps for your Laravel application. Mix supports several common CSS and JavaScript pre-processors. If you've ever been confused about how to get started with module bundling and asset compilation, you will love Laravel Mix!

Using NPM scripts

This is are ready to use command to compile the Attrax assets.

Task Description
npm run dev npm run dev creates the "/dist" directory with compiled files.
npm run watch The npm run watch command will continue running in your terminal and watch all relevant files for changes. Webpack will then automatically recompile your assets when it detects a change.
npm run hot Hot Module Replacement (or Hot Reloading) allows you to, not just refresh the page when a piece of JavaScript is changed, but it will also maintain the current state of the component in the browser.
npm run production npm run production will run all Mix tasks and minify output that which is ready for use for production.
