Animations

We use a 3rd plugin from animate.css to create animated effects from this template. More info about the list of available animations can be found on the official page of animate.css. The following are some examples of animations that are ready for you to use.

Utility Class Description
animate fadeInUp Make the animation effect fade up slowly
animate fadeInDown Make the animation effect fade down slowly
animate fadeInLeft Make the animation effect fade left slowly
animate fadeInRight Make the animation effect fade right slowly
