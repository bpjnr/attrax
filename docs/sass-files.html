SASS Files

In the sass directory there are several separate files and directories. File separation is based on the purpose and function of each code.

The following is a detailed description of each file and directory.

Components Directory

This directory contains template component files, either custom components or modifications to the Bootstrap component.

Fonts Directory

This directory contains files to import fonts used by templates.

Utilities Directory

This directory contains utility template files, either custom utilities or modifications to Bootstrap utilities.

_fonts.scss

This file is used to import fonts used in the template.

_variables.scss

This file contains a list of variables used in all components of the template and configuration variables in Bootstrap.

main.scss

This file is the main file that imports all the files above.
