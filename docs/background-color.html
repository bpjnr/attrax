Background Color

This utility serves to provide background color on an element. The colors that you can use are listed in the $main-colors variable located in the /sass/_variables.scss file. To see the available color display you can access the color.html page. Please add a new color to the file to be able to use more colors.

Utility Class Description
bg-color-1 Will produce the background color #858ba5
bg-color-2 Will produce the background color #8a95a9
bg-color-3 Will produce the background color #575e7d
bg-color-4 Will produce the background color #1c1e2b

You can also set the background color only for certain breakpoints with the format bg-color-{breakpoint}-{colorIndex}.

Utility Class Description
bg-color-sm-1 Will produce the background color #858ba5
bg-color-md-1 Will produce the background color #8a95a9
bg-color-lg-1 Will produce the background color #575e7d
bg-color-xl-1 Will produce the background color #1c1e2b
