File Contents

After you download this template, you will find 3 files in it.

If you want to directly use this template without making modifications you can immediately use the file in the compiled directory. In that directory there is a file that has been compiled and terminated so that it is ready to be used directly.

But if you want to make modifications, add new components, or use your own build tools it's better to use files in the source directory.
