JS Contents

All of these javascript template files are located in the /src/js/ directory. Each code is made separately based on certain components or modules to make modifications easier.

If you want to create a javascript code for a new component you can create a file in this directory and import the file in /src/js/app.js, so that the file is compiled by the webpack.
